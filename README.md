# Charts-中兴华易应用平台

> Charts-个人应用平台

## 项目安装运行

#### 安装依赖

``` bash
# install dependencies
npm install
```
#### 运行项目

```

# serve with hot reload at localhost:8080
npm run dev
```
#### 打包

```

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
